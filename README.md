# OZON-PILOT Release

This repository contains only the packaged public release artifacts for OZON-PILOT.

## Included

- obfuscated application binaries
- required runtime dependencies
- bundled plugin/runtime assets needed by the packaged app

## Not Included

- source code
- build scripts
- debug symbols
- obfuscation mapping files

## Run

Extract the folder and start `OZON-PILOT.exe`.

If WebView2 is missing on the target machine, run `Webview\Setup.exe` or let the installer/runtime bootstrap handle it.

## Notes

This public repository is intended for distribution of release artifacts only.
